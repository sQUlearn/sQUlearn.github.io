class squlearn.encoding_circuit.TranspiledEncodingCircuit(encoding_circuit: EncodingCircuitBase, backend: Backend, transpile_func: Callable | None = None, **kwargs)ÔÉÅ

Class for generated a Encoding Circuit with a transpiled circuit.

Example:

from squlearn.encoding_circuit import TranspiledEncodingCircuit,ChebyshevRx
from qiskit_ibm_runtime.fake_provider import FakeManilaV2

fm = TranspiledEncodingCircuit(ChebyshevRx(3,1),backend=FakeManilaV2(),initial_layout=[0,1,4])
fm.draw("mpl")

  • transpile_func (Union[Callable,None]) ‚Äì Optional function for transpiling the circuit. First argument is the circuit, second the backend. If no function is specified, Qiskit‚Äôs transpile function is used.
