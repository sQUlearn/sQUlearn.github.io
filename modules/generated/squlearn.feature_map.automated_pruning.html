squlearn.feature_map.automated_pruningÔÉÅ

squlearn.feature_map.automated_pruning(feature_map: FeatureMapBase, executor: Executor, n_sample: int = 1, pruning_thresh: float = 1e-10, x_lim: Optional[Tuple[float, float]] = None, p_lim: Optional[Tuple[float, float]] = None, x_val: Optional[Tuple[float, float]] = None, p_val: Optional[Tuple[float, float]] = None, verbose: int = 1, seed: Union[None, int] = None) → PrunedFeatureMapÔÉÅ

Function for automated pruning of the parameters in the inputted parameterized quantum circuit.

The algorithms for the automated pruning is based on https://doi.org/10.1103/PRXQuantum.2.040309.

Parameters:

  • feature_map (FeatureMapBase) ‚Äì Parameterized quantum circuit that will be pruned. Has to be in the feature_map format of quantum_fit!

  • QI (QuantumInstance) ‚Äì Quantum Instance for evaluating the Quantum Fisher Information Matrix

  • n_sample (int) ‚Äì Number of random parameter values and input data that is generated. (if x_val=None and p_val=None, the number of evaluated Fisher matrices is n_sample*n_sample)

  • pruning_thresh (float) ‚Äì Threshold for pruning, eigenvalues lower that value are considered to be redundant.

  • x_lim (Union[Tuple[float, float],None]) ‚Äì Limits for the random input data values; default limits are \((-\pi,\pi)\).

  • p_lim (Union[Tuple[float, float],None]) ‚Äì Limits for the random parameter values; default limits are \((-\pi,\pi)\).

  • x_val (Union[Tuple[float, float],None]) ‚Äì Array with input data values. If specified, no random input data is generated.

  • p_val (Union[Tuple[float, float],None]) ‚Äì Array with parameter values. If specified, no random parameter values are generated.

  • verbose (int) ‚Äì Verbosity of the algorithm, as default only the redundant parameters are printed.

  • seed (int) ‚Äì Seed for the random input data and parameters generation.

Returns:

Pruned feature_map as a PrunedFeatureMap class object.
