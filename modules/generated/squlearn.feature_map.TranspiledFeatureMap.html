squlearn.feature_map.TranspiledFeatureMap

class squlearn.feature_map.TranspiledFeatureMap(feature_map: FeatureMapBase, backend: Backend, transpile_func: Optional[Callable] = None, **kwargs)

Class for generated a Feature Map with a transpiled circuit.

Example:

from squlearn.feature_map import TranspiledFeatureMap,ChebRx
from qiskit.providers.fake_provider import FakeManilaV2

fm = TranspiledFeatureMap(ChebRx(3,1),backend=FakeManilaV2(),initial_layout=[0,1,4])
fm.draw()

(Source code, png, hires.png, pdf)

Parameters:

  • feature_map (FeatureMapBase) – Feature map to be transpiled.

  • backend (Backend) – Backend used for the transpilation.

  • transpile_func (Union[Callable,None]) – Optional function for transpiling the circuit. First argument is the circuit, second the backend. If no function is specified, Qiskit’s transpile function is used.

  • kwargs – Additional arguments for Qiskit’s transpile function.

draw(output: Optional[str] = None, feature_label: str = 'x', parameter_label: str = 'p', decompose: bool = False, **kwargs) → None

  • parameter_label (str) – Label for the parameter vector (default:”p”).

  • decompose (bool) – If True, the circuit is decomposed before printing (default: False).

  • kwargs – Additional arguments from Qiskit’s QuantumCircuit.draw() function.
