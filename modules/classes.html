Circuit Design

Feature Maps

feature_map.YZ_CX_FeatureMap

Creates the YZ-CX Feature Map from Reference https://arxiv.org/abs/2108.01039v3

feature_map.HighDimFeatureMap

The high-dimensional feature map from Reference https://doi.org/10.1103/PhysRevA.106.042431

feature_map.QEKFeatureMap

Creates the QEK feature map as presented in Reference http://arxiv.org/pdf/2105.02276v1

feature_map.ChebyshevTower

A feature-map that is based on the Chebyshev Tower encoding.

feature_map.ChebPQC

Chebyshev Feature Map from Reference https://arxiv.org/abs/2306.01639

feature_map.HZCRxCRyCRz

Feature map with HZ encoding followed by controlled Rx, Ry Rz rotations.

feature_map.ChebRx

Simple Chebyshev feature map build from Rx gates

feature_map.ParamZFeatureMap

Parameterized ZFeatureMap with optional CNOT gates between the default layers.

feature_map.QiskitZFeatureMap

Wrapper to use Qiskit's ZFeatureMap

feature_map.QiskitFeatureMap

Wrapper to create sQUlearn feature maps from the Qiskit circuit library.

Feature Map Tools

feature_map.feature_map_base.FeatureMapBase

Feature map base class

feature_map.PrunedFeatureMap

Class for pruning redundant parameter of feature maps.

feature_map.LayeredFeatureMap

A class for a simple creation of layered feature maps.

feature_map.FeatureMapDerivatives

Class for automatic differentiation of feature maps.

feature_map.TranspiledFeatureMap

Class for generated a Feature Map with a transpiled circuit.

feature_map.automated_pruning

Function for automated pruning of the parameters in the inputted parameterized quantum circuit.

feature_map.pruning_from_QFI

Algorithm for determining the redundant parameters from the Quantum Fischer Information.

Operators

expectation_operator.SinglePauli

Single Pauli operator of a single Qubit.

expectation_operator.SummedPaulis

Operator for summation of single Pauli operators.

expectation_operator.SingleProbability

Operator for measuring the probability of being in state 0 or 1 of a specified qubit.

expectation_operator.SummedProbabilities

Operator for summing single Qubit probabilities of binary states.

expectation_operator.IsingHamiltonian

Implementation of Ising type Hamiltonians.

expectation_operator.CustomExpectationOperator

Class for evaluating the expectation value of a custom operator.
